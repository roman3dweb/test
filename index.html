Свободное программное обеспечение для инженеров

Данный сайт предназначен для инженеров и студентов технических специальностей. Материалы сайта содержат описания и примеры работы со свободными программами. Сайт не предназначен для профессиональных програмистов. Цель сайта научить использовать программы с открытым исходным кодом тех, для кого плохо знаком с программированием и linux

В настоящее время англоязычным научным сообществом создано множество расчетных и графических программ с открытым исходным кодом. Данные программы доступны для коммерческой деятельности, т.е согласно открытой лицензии допускается пользоваться результами, но запрещается распостранять программы в коммерческих целях.
